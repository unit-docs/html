# HTML HOME
Belajar Mudah dengan HTML "Coba Sendiri"
Dengan editor, Anda dapat mengedit kode HTML dan melihat hasilnya
contoh :
```sh
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
</body>
</html> 
```
