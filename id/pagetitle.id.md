# Judul Halaman HTML
Setiap halaman web harus memiliki judul halaman untuk menggambarkan arti halaman tersebut.
Elemen `<title>` menambahkan judul ke halaman Anda:

Contoh : 
```sh
<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
</head>
<body>
```
##### The content of the document......
```sh
</body>
</html>
```
