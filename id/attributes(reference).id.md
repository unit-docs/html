# Referensi Atribut HTML

Tabel di bawah ini mencantumkan semua atribut HTML dan elemen apa saja yang dapat digunakan di dalamnya:

| Attribut | Gunakan Untuk | Deskripsi |
| ----------- | ----------- | ----------- |
| accept | `<input>` | Menentukan jenis file yang diterima server (hanya untuk type="file") |
| accept-charset	 | `<form>` | Menentukan pengkodean karakter yang akan digunakan untuk pengiriman formulir |
| accesskey	 | `Global Attributes` | Menentukan tombol pintas untuk mengaktifkan/memfokuskan elemen |
| action	 | `<form>` | Menentukan ke mana harus mengirim data formulir ketika formulir dikirimkan |
| align	 | Not supported in HTML 5.	 | Menentukan perataan menurut elemen di sekitarnya. Gunakan CSS sebagai gantinya |
| alt | `<area>`, `<img>`, `<input>`	 | Menentukan teks alternatif saat elemen asli gagal ditampilkan |
| async	 | `<script>` | Menentukan bahwa skrip dieksekusi secara asinkron (only for external scripts) |
| autocomplete	 | `<form>`, `<input>` | Menentukan apakah elemen `<form>` atau `<input>` harus diaktifkan pelengkapan otomatisnya |
| autofocus	 | `<button>`, `<input>`, `<select>`, `<textarea>`	 | Menentukan bahwa elemen harus secara otomatis mendapatkan fokus saat halaman dimuat |
| autoplay	 | `<audio>`, `<video>` | Menentukan bahwa audio/video akan mulai diputar segera setelah siap |
| bgcolor	 | Not supported in HTML 5.	 | Menentukan perataan menurut elemen di sekitarnya. Gunakan CSS sebagai gantinya |
| border |  Not supported in HTML 5. | Menentukan perataan menurut elemen di sekitarnya. Gunakan CSS sebagai gantinya |
| charset	 | `<meta>`, `<script>`	 | Specifies the character encoding |
| checked	 | `<input>` | Menentukan bahwa elemen `<input>` harus dipilih sebelumnya saat halaman dimuat (for type="checkbox" or type="radio")  |
| cite	 | `<blockquote>`, `<del>`, `<ins>`, `<q>`	 | Menentukan URL yang menjelaskan kutipan/teks yang dihapus/disisipkan |
| class	 | Global Attributes	 | Menentukan satu atau beberapa nama kelas untuk sebuah elemen (merujuk ke kelas dalam style sheet) |
| color	 | Not supported in HTML 5.	 | Menentukan perataan menurut elemen di sekitarnya. Gunakan CSS sebagai gantinya |
| cols	 | `<textarea>` | Menentukan lebar area teks yang terlihat |
| colspan	 | `<td>`, `<th>`	 | Menentukan jumlah kolom yang harus direntang oleh sel tabel |
| content	 | `<meta>` | Memberikan nilai yang terkait dengan atribut http-equiv atau nama |
| contenteditable	 | Global Attributes	 | Specifies whether the content of an element is editable or not |
| controls	 | `<audio>`, `<video>`	 | Menentukan kontrol audio/video yang harus ditampilkan (seperti tombol putar/jeda, dll.) |
| coords	 | `<area>` | Menentukan koordinat area |
| data	 | `<object>` | Menentukan URL sumber daya yang akan digunakan oleh objek |
| data-*	 | Global Attributes	 | Digunakan untuk menyimpan data khusus yang bersifat pribadi pada halaman atau aplikasi |
| datetime	 | `<del>`, `<ins>`, `<time>`	 | Menentukan tanggal dan waktu |
| default	 | `<track>` | Menentukan bahwa track akan diaktifkan jika preferensi pengguna tidak menunjukkan bahwa track lain akan lebih sesuai |
| defer	 | `<script>` | Menentukan bahwa skrip dijalankan ketika halaman telah selesai diurai (hanya untuk external scripts) |
| dir	 | Global Attributes	 | Specifies the text direction for the content in an element |
| dirname	 | `<input>`, `<textarea>` | Menentukan arah teks yang akan dikirimkan |
| disabled	 | `<button>`, `<fieldset>`, `<input>`, `<optgroup>`, `<option>`, `<select>`, `<textarea>`	 | Menentukan bahwa elemen/kelompok elemen yang ditentukan harus dinonaktifkan |
| download	 | `<a>`, `<area>` | Menentukan bahwa target akan diunduh saat pengguna mengeklik hyperlink |
| draggable	 | Global Attributes	 | Menentukan apakah sebuah elemen dapat digeser atau tidak |
| enctype	 | `<form>` | Menentukan bagaimana data formulir harus dikodekan saat mengirimkannya ke server (hanya untuk method="post") |
| enterkeyhint	 | Global Attributes	 | Menentukan teks tombol enter pada keybord virtual |
| for	 | `<label>`, `<output>`	 | Menentukan elemen formulir mana yang diikat oleh label/perhitungan |
| form	 | `<button>`, `<fieldset>`, `<input>`, `<label>`, `<meter>`, `<object>`, `<output>`, `<select>`, `<textarea>`	 | Menentukan nama formulir yang dimiliki elemen |
| formaction	 | `<button>`, `<input>`	 | Menentukan ke mana harus mengirim data formulir ketika formulir dikirimkan. Hanya untuk tipe = "submit" |
| headers	 | `<td>`, `<th>`	 | Menentukan satu atau beberapa sel tajuk yang terkait dengan sel |
| height	 | `<canvas>`, `<embed>`, `<iframe>`, `<img>`, `<input>`, `<object>`, `<video>`	 | Menentukan ketinggian elemen |
| hidden	 | Global Attributes	 | Menentukan bahwa suatu elemen belum, atau tidak lagi, relevan |
| high	 | `<meter>` | Menentukan rentang yang dianggap sebagai nilai tinggi |
| href	 | `<a>`, `<area>`, `<base>`, `<link>`	 | Menentukan URL halaman yang dituju oleh tautan |
| hreflang	 | `<a>`, `<area>`, `<link>`	 | Menentukan bahasa dokumen yang ditautkan |
| http-equiv	 | `<meta>` | Menyediakan header HTTP untuk informasi/nilai atribut konten |
| id	 | Global Attributes	 | Menentukan id unik untuk sebuah elemen |
| inert	 | Global Attributes | Menentukan bahwa browser harus mengabaikan bagian ini |
| inputmode	 | Global Attributes	 | Menentukan mode keyboard virtual |
| ismap	 | `<img>` | Menentukan gambar sebagai peta gambar sisi server |
| kind	 | `<track>` | Menentukan jenis track teks |
| label	 | `<track>`, `<option>`, `<optgroup>`	 | Menentukan judul track teks |
| lang	 | Global Attributes	 | Menentukan bahasa konten elemen |
| list	 | `<input>` | Merujuk ke elemen `<datalist>` yang berisi opsi yang telah ditentukan sebelumnya untuk elemen `<input>` |
| loop	 | `<audio>`, `<video>`	 | Menentukan bahwa audio/video akan dimulai dari awal lagi, setiap kali selesai |
| low	 | `<meter>` | Menentukan kisaran yang dianggap sebagai nilai rendah |
| max	 | `<input>`, `<meter>`, `<progress>`	 | Menentukan nilai maksimum |
| maxlength	 | `<input>`, `<textarea>`	 | Menentukan jumlah karakter maksimum yang diizinkan dalam sebuah elemen |
| media	 | `<a>`, `<area>`, `<link>`, `<source>`, `<style>`	 | Menentukan media/perangkat apa yang dioptimalkan untuk dokumen yang ditautkan |
| method	 | `<form>` | Menentukan metode HTTP yang akan digunakan saat mengirim data formulir |
| min	 | `<input>`, `<meter>`	 | Menentukan nilai minimum |
| multiple	 | `<input>`, `<select>`	 | Menentukan bahwa pengguna dapat memasukkan lebih dari satu nilai |
| muted	 | `<video>`, `<audio>`	 | Menentukan bahwa output audio dari video harus dibisukan |
| name	 | `<button>`, `<fieldset>`, `<form>`, `<iframe>`, `<input>`, `<map>`, `<meta>`, `<object>`, `<output>`, `<param>`, `<select>`, `<textarea>`	 | Menentukan nama elemen |
| novalidate	 | `<form>` | Menentukan bahwa formulir tidak boleh divalidasi saat dikirimkan |
| onabort	 | `<audio>`, `<embed>`, `<img>`, `<object>`, `<video>`	 | Skrip yang akan dijalankan saat dibatalkan |
| onafterprint	 | `<body>`	 | Skrip yang akan dijalankan setelah dokumen dicetak |
| onbeforeprint	 | `<body>` | Skrip yang akan dijalankan sebelum dokumen dicetak |
| onbeforeunload	 | `<body>`  | Skrip yang akan dijalankan saat dokumen akan dibongkar |
| onblur	 | All visible elements.	 | Skrip yang akan dijalankan saat elemen kehilangan fokus |
| oncanplay	 | `<audio>`, `<embed>`, `<object>`, `<video>`	 | Skrip yang akan dijalankan ketika file siap untuk mulai diputar (ketika sudah cukup buffer untuk memulai) |
| oncanplaythrough	 | `<audio>`, `<video>`	 | Skrip yang akan dijalankan ketika file dapat diputar hingga akhir tanpa jeda untuk buffering |
| onchange	 | All visible elements.	 | Skrip yang akan dijalankan saat nilai elemen diubah |
| onclick	 |  All visible elements. | Skrip yang akan dijalankan saat elemen diklik |
| oncontextmenu	 | All visible elements.	 | Skrip yang akan dijalankan ketika menu konteks dipicu |
| oncopy	 | All visible elements.	 | Skrip yang akan dijalankan saat konten elemen disalin |
| oncuechange	 | `<track>` | Skrip yang akan dijalankan saat isyarat berubah dalam elemen `<track>` |
| oncut	 | All visible elements.	 | Skrip yang akan dijalankan saat konten elemen dipotong |
| ondblclick	 | All visible elements.	 | Skrip yang akan dijalankan saat elemen diklik dua kali |
| ondrag	 | All visible elements.	 | Skrip yang akan dijalankan saat elemen digeser |
| ondragend	 | All visible elements.	 | Skrip yang akan dijalankan di akhir operasi geser |
| ondragenter	 | All visible elements.	 | Skrip yang akan dijalankan saat elemen digeser ke target jatuh yang valid |
| ondragleave	 | All visible elements.	 | Skrip yang akan dijalankan ketika sebuah elemen meninggalkan target jatuh yang valid |
| ondragover	 | All visible elements.	 | Skrip yang akan dijalankan saat elemen digeser di atas target jatuh yang valid |
